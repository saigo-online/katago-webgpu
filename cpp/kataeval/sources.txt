# kataeval — minimal KataGo source set for neural-net evaluation (WebGPU backend).
# This is the single source of truth for the dependency's contents: one path per
# line, relative to cpp/. Read by both kataeval/CMakeLists.txt (native) and
# scripts/build-eval.sh (WASM). No search / GTP / command/ — just model load,
# input features, and the WebGPU forward pass.
#
# Dual backend in one binary: kataeval/backend_gpu.cpp (WebGPU, C++20) and
# kataeval/backend_cpu.cpp (Eigen, needs -DUSE_EIGEN_BACKEND + eigen3 headers)
# are renamed into separate namespaces; kataeval/dispatch.cpp routes NeuralNet::
# to one, trying WebGPU then falling back to CPU. backend_gpu.cpp #includes
# webgpubackend.cpp (so don't list it directly); likewise backend_cpu.cpp.
